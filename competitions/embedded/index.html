eCTF

An attack-and-defend exercise for designing secure embedded systems.

Collegiate eCTF 2018

Overview

New! We are gearing-up now for the 2018 Collegiate eCTF! Check back here for more information soon!

MITRE's eCTF (embedded capture-the-flag) is an embedded security competition that puts participants through the experience of trying to create a secure system and then learning from their mistakes. The main target is a real physical embedded device, which opens the scope of the challenge to include physical/proximal access attacks.
